Everything agents need, nothing they don't

A unified SDK replacing fragmented wallet libraries, swap APIs, bridge services, and identity systems.

🔒

Session-Scoped Wallets

Create wallets that auto-expire and enforce spending limits. Private keys are AES-256 encrypted and never enter the agent's context window. Set contract allowlists to control what your agent can interact with.

🔁

TaikoSwap Integration

Swap any token through TaikoSwap, the native DEX on Taiko. Get quotes, set slippage tolerance, and execute swaps in a single function call. Built on verified UniswapV2 contracts.

🔗

L1/L2 Bridging

Bridge ETH between Ethereum and Taiko through the native bridge contracts. Estimate fees, track bridge status, and manage cross-chain operations programmatically.

💰

Token Operations

Query native ETH and ERC-20 balances across chains. Transfer tokens with a single call. Supports all standard tokens on Taiko mainnet.

🤖

Onchain Agent Identity

Register your agent onchain with metadata and capabilities. Build verifiable reputation through success/failure tracking. Discover other agents by owner or ID. Inspired by ERC-8004.

⚡

Chain Queries & RPC

Query blocks, gas prices, transactions, and read arbitrary smart contracts. Built on Taiko's dedicated RPC infrastructure with connection pooling.

Up and running in 5 lines

Framework-agnostic TypeScript SDK. Works with Claude, LangChain, CrewAI, or any agent framework.

agent.ts 
import { TaikoAgentKit } from "@taiko/agent-toolkit";

// Create a session with guardrails
const kit = TaikoAgentKit.create({
  chainId: 167000,
  walletConfig: {
    spendingLimitEth: "0.5",
    ttlSeconds: 3600,
  },
});

// Check balance
const balance = await kit.getBalance();

// Swap ETH for TAIKO via TaikoSwap
await kit.swap({
  fromToken: "ETH",
  toToken: "TAIKO",
  amount: "0.1",
});

// Bridge to Ethereum L1
await kit.bridge({ amount: "0.5", toChain: 1 });

// Register agent identity onchain
await kit.registerAgent({
  name: "my-defi-agent",
  capabilities: ["swap", "bridge", "monitor"],
});

Manage wallets, check balances, get swap quotes, and query chain state from your terminal.

Agent-First Security

Designed for agent-specific threat models from day one.

🔐 Encrypted Keys

AES-256-CBC + PBKDF2 (100k iterations). Keys never enter the agent context.

💰 Spending Limits

Configurable ETH caps per session. Transactions exceeding limits are rejected.

⏰ Auto-Expiry

Wallets have a TTL. Expired wallets cannot sign transactions.

📜 Contract Allowlists

Restrict which contracts the agent can interact with. Block rogue calls.

Modular Architecture

Framework-agnostic core with thin adapter layers.

@taiko/agent-toolkit             // Core SDK + CLI
  |
  +-- wallet/                      // Session wallets + encryption
  +-- tokens/                      // ERC-20 balances & transfers
  +-- swap/                        // TaikoSwap (UniswapV2) integration
  +-- bridge/                      // Taiko native bridge (L1 <-> L2)
  +-- identity/                    // Onchain agent registry & reputation
  +-- rpc/                         // Chain queries & contract reads
  +-- config/                      // Verified chain configs & addresses
